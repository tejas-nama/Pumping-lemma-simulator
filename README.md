## 🎯 What It Does

This tool walks you through a complete formal proof that a given language is not regular or not context-free, using the Pumping Lemma. Each proof is broken into 5 guided interactive steps:

| Step | Description |
|------|-------------|
| 1 | Theorem — Statement of the Pumping Lemma |
| 2 | Choose string — Pick a string s ∈ L with |s| ≥ p |
| 3 | Decompose — Visualize the xyz / uvxyz split with colour-coded cells |
| 4 | Pump it! — Change the pump value i (0–4) and see the result live |
| 5 | Verdict — Complete formal proof summary with QED |

## 📖 Theory Background

### Pumping Lemma for Regular Languages

Theorem: If L is regular, ∃ p ≥ 1 such that for every s ∈ L with |s| ≥ p, s = xyz where:
- |y| ≥ 1
- |xy| ≤ p
- ∀ i ≥ 0 : xyⁱz ∈ L

Proof strategy: Assume L is regular → pick s ∈ L with |s| ≥ p → show for ANY decomposition, some xyⁱz ∉ L → contradiction → L is not regular □

### Pumping Lemma for Context-Free Languages

Theorem: If L is CF, ∃ p ≥ 1 such that for every s ∈ L with |s| ≥ p, s = uvxyz where:
- |vy| ≥ 1
- |vxy| ≤ p
- ∀ i ≥ 0 : uvⁱxyⁱz ∈ L
